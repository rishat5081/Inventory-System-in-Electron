## Inventory System in Electron

The inventory system contains the following features

1. Sales
2. Purchase
3. Record
4. Inventory
5. Employees Records
6. Employee Sales

Following are the frameworks and libraries are used:

- Node 17.3.0
- Electron 16.0.5
- NPM 8.3.0
- Sequelize 7.0.0
- Sequelize-cli
- MYSQL 8
